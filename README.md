🔍 C Metin Arama Motoru (Text Search Engine)



Bu proje, metin dosyaları içerisinde belirli anahtar kelimeleri arayan, bunların frekansını (tekrar sayısı) ve satır numaralarını raporlayan bir C programıdır. İzmir Bakırçay Üniversitesi Bilgisayar Mühendisliği Bölümü bünyesindeki proje isterlerine uygun olarak geliştirilmiştir.



✨ Özellikler



Dinamik Arama: Kullanıcıdan alınan herhangi bir anahtar kelimeyi dosya içerisinde arar.



Detaylı Raporlama: Kelimenin geçtiği tüm satır numaralarını tek tek listeler.



İstatistiksel Veri: Aramanın sonunda kelimenin toplamda kaç kez geçtiğini hesaplar.



Modüler Yapı: Temiz ve okunabilir C kodu yapısı ile geliştirilmiştir.



🚀 Başlangıç



Projeyi yerel makinenizde çalıştırmak için aşağıdaki adımları izleyebilirsiniz:



Gereksinimler



Bir C derleyicisi (GCC, Clang veya MSVC).



VS Code veya benzeri bir editör.



Kurulum ve Çalıştırma



Bu depoyu bilgisayarınıza indirin.



Kodun bulunduğu klasörde terminali açın.



Kodu derleyin:



gcc main.c -o arama\_motoru





Programı çalıştırın:



./arama\_motoru





📂 Kullanım



Program çalıştırıldığında:



Aranacak kelimeyi girin.



Aramanın yapılacağı dosya adını girin (örn: test.txt).



Program sonuçları terminal üzerinde raporlayacaktır.



Önemli: Metin dosyasının .exe ile aynı klasörde olduğundan emin olun.



🛠️ Teknik Detaylar



Dil: C



Kütüphaneler: stdio.h, string.h, stdlib.h



Algoritma: Satır bazlı okuma (fgets) ve string eşleştirme (strstr) yöntemleri kullanılmıştır.
